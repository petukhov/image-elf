# Image Elf

> Easily generate images of any size.

Set your desired dimensions, select the image format, and download your creation instantly!

## Usage

Install Packages

```bash
pnpm i
```

Run Development

```bash
pnpm run dev
```

Build

```bash
pnpm run build
```
